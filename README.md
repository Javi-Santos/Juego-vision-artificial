# Juego-vision-artificial
PROYECTO JUEGO GUITARRISTA DE COLORES

Instrucciones:
1. Ensayar webcam y que no capte colores rojos o verdes en un inicio rapido para ver el interfaz completo del juego
2. Correr el codigo y mostrar objeto de color rojo en webcam para iniciar el juego
3. Manejar el personaje con el objeto verde en la webcam y hacer que caigan las notas encima de él.
4. Las notas que caigan al suelo marcarán puntos en contra.
5. Conseguir 5 puntos a favor o en contra, o retirarse pulsando la letra "Q".
6. Mostrar objeto rojo en webcam para finalizar y salir del juego.
